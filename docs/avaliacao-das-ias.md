# Avaliação das IAs Testadas no Projeto Labirinto

Este documento reúne as inteligências artificiais testadas durante o desenvolvimento do projeto Labirinto. A avaliação considera para que cada IA foi usada, o que ela entregou, pontos positivos, pontos negativos e nota final.

## 1. Lovable

**Para que usei:** teste de criação do site.

**O que ela fez:** criou uma versão inicial do site, mas ainda ficou um pouco distante do resultado esperado. Precisou de bastante orientação passo a passo.

**Pontos positivos:** foi uma das IAs que mais chegou perto da ideia do site nas primeiras tentativas.

**Pontos negativos:** não entendeu tudo de primeira, teve problemas de organização e design e exigiu muitos ajustes.

**Nota:** 7,5/10.

**Observação:** foi útil para testar a ideia inicial, mas não entregou o sistema completo sem ajuda.

---

## 2. Ferramenta AI

**Para que usei:** pesquisa e descoberta de outras IAs para tarefas específicas, como geração de imagem.

**O que ela fez:** indicou diferentes ferramentas que poderiam ajudar no projeto.

**Pontos positivos:** fácil de usar, dinâmica e boa para encontrar opções de IA.

**Pontos negativos:** nenhum ponto negativo importante foi identificado.

**Nota:** 10/10.

**Observação:** foi muito útil para descobrir novas ferramentas, principalmente para quem ainda está explorando opções de IA.

---

## 3. Prompt Genie

**Para que usei:** tentativa de melhorar o prompt usado para criar o site.

**O que ela fez:** não ajudou muito no resultado final.

**Pontos positivos:** não teve pontos positivos importantes na experiência.

**Pontos negativos:** interface em inglês, difícil de usar e de entender. Acabou tomando tempo e não resolveu o problema.

**Nota:** 3/10.

**Observação:** não foi uma ferramenta boa para esse projeto.

---

## 4. Free AI Article Generator

**Para que usei:** tentativa de gerar o corpo do artigo do projeto.

**O que ela fez:** não gerou um resultado útil para o trabalho.

**Pontos positivos:** a interface parecia dinâmica, mas não foi clara o suficiente.

**Pontos negativos:** não respondeu bem ao pedido e não entregou o que era necessário.

**Nota:** 4/10.

**Observação:** não pretendo usar novamente para esse tipo de tarefa.

---

## 5. Bubble.io

**Para que usei:** tentativa de criar uma versão do site.

**O que ela fez:** criou mais uma estrutura visual do que um site pronto com código. Funcionou de forma parecida com um construtor visual, lembrando o Figma.

**Pontos positivos:** muito bom para criar e testar layout, CSS e estrutura visual.

**Pontos negativos:** é complexo no começo e exige um tempo de prática para entender melhor.

**Nota:** 5/10.

**Observação:** pode ser útil para design, mas não foi a melhor opção para criar o sistema completo.

---

## 6. V0 App

**Para que usei:** criação de uma versão do site.

**O que ela fez:** conseguiu gerar um site com base no prompt enviado.

**Pontos positivos:** fácil de usar, dinâmica e com interface simples de entender.

**Pontos negativos:** teve dificuldade para entender completamente requisitos mais complexos do sistema.

**Nota:** 7/10.

**Observação:** foi uma boa ferramenta e pode ser usada novamente com prompts mais bem organizados.

---

## 7. Gamma.app

**Para que usei:** tentativa de criação do site.

**O que ela fez:** criou algo mais parecido com um resumo ou apresentação sobre o site, explicando o que ele é e para que serve.

**Pontos positivos:** ajudou a entender melhor a ideia do sistema e pode servir para estudar o projeto com mais profundidade.

**Pontos negativos:** não criou o site funcional.

**Nota:** 7/10.

**Observação:** é mais útil para explicação e apresentação do que para desenvolvimento direto do sistema.

---

## Conclusão

Durante os testes, algumas IAs foram melhores para criar estrutura visual, outras para estudar a ideia do projeto e outras para encontrar ferramentas. Nenhuma resolveu tudo sozinha, mas cada uma ajudou em uma parte do desenvolvimento do Labirinto.

As ferramentas mais úteis até agora foram a **Ferramenta AI**, por ajudar a encontrar recursos, e o **V0 App**, por conseguir gerar uma versão funcional do site a partir do prompt.
