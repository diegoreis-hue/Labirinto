# Labirinto

Sistema web para armazenamento, organização e controle de materiais dos laboratórios **Labirinto**, **Labin Maker** e **D03**.

O projeto tem como foco ajudar na gestão do inventário dos laboratórios, permitindo cadastrar materiais, organizar itens por pastas, controlar empréstimos, aprovar solicitações, registrar devoluções e visualizar quem está com cada item emprestado.

## Objetivo

Criar um site simples, bonito e funcional para facilitar o controle dos materiais dos laboratórios. A ideia é evitar perda de equipamentos, melhorar a organização e deixar claro quais itens estão disponíveis, emprestados ou aguardando aprovação.

## Principais funções

- Login com diferentes tipos de usuário.
- Área de administrador, colaborador e usuário comum.
- Cadastro de pastas e itens do inventário.
- Visualização de quantidade total, disponível e emprestada.
- Sistema de solicitação de empréstimo.
- Aprovação ou recusa de pedidos.
- Sistema de devolução parecido com o fluxo de empréstimo.
- Contagem de itens emprestados atualizada automaticamente.
- Clique na quantidade emprestada para ver quem está com o item.
- Aba de responsáveis com contatos importantes.
- Notificações por e-mail para avisos importantes.
- Acesso a outros sistemas, como o Labin Maker, por seleção de logo.

## Identidade visual

O estilo visual do sistema deve seguir uma ideia tecnológica, usando principalmente **preto e dourado**, com aparência inspirada em placa-mãe, processadores, circuitos e robótica.
